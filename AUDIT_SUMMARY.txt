================================================================================
TECHNICAL DETAILS
================================================================================

ROOT CAUSES:
1. fetch() calls without AbortSignal.timeout()
2. User abort signals not composed with timeout signals
3. Missing timer cleanup in finally blocks
4. Promise.all() without overall timeout wrapper
5. No timeout context passed through async chains

PATTERN VIOLATIONS:
  ✗ Missing: const controller = new AbortController()
  ✗ Missing: signal: AbortSignal.timeout(ms)
  ✗ Missing: finally { clearTimeout(id) }
  ✗ Missing: signal?.addEventListener('abort', ...)

GOOD PATTERNS FOUND:
  ✓ ollama.ts: Proper timeout + idle timeout + circuit breaker + retry
  ✓ wayfayer.ts screenshot methods: Some have proper timeout composition
  ✓ infrastructure.ts: Individual health checks have timeout
  ✓ nativeMacBridge.ts health check: Has proper timeout
