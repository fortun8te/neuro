with production-grade features for research, export, and academic workflows.

DESIGN DECISIONS
─────────────────────────────────────────────────────────────────────────────

1. CITATION NUMBERING
   - Superscript in circular badges for visual appeal
   - Index-based (not hash-based) for consistency
   - Persistent across expansions/sorts

2. METADATA DISPLAY
   - Expandable cards to keep compact by default
   - Fetch date and content type optional fields
   - Relevance score as visual bar with percentage

3. EXPORT OPTIONS
   - Copy-to-clipboard default for speed (no dialog)
   - Download button for bibliography (creates file)
   - Format picker in consistent menu structure

4. CITATION FORMATS
   - APA: Most common in academic writing
   - MLA: Standard for humanities/literature
   - Chicago: Used in history and journalism
   - Simple implementation, no library dependencies

5. ORGANIZATION
   - Sort: 3 modes (domain, relevance, date)
   - Group: By domain or flat (not redundant)
   - Keyboard-accessible filter menu

INTEGRATION EXAMPLES
─────────────────────────────────────────────────────────────────────────────

Example 1: Basic Usage
  import { SourcesList } from './components/SourcesList';
  
  <SourcesList sources={sources} maxVisible={3} variant="modal" />

Example 2: With Research Findings
  import { extractSourcesFromFindings } from './utils/sourceExtractor';
  
  const sources = extractSourcesFromFindings(researchFindings);
  <SourcesList sources={sources} sortBy="relevance" />

Example 3: Export Flow
  import { downloadSources } from './utils/sourceExporter';
  
  onClick={() => downloadSources(sources, 'Bibliography')}

Example 4: Inline Citations in Text
  import { injectCitationsIntoText } from './utils/inlineCitations';
  
  const { citations } = injectCitationsIntoText(text, sources);
