     return fallbackValue;
   }

   import type { Type } from './module';  // Safe, tree-shakes

   - Implement feature detection helpers
   - Provide fallback implementations

   - Help with debugging module loading issues
   - Track which features are unavailable
   - Monitor for missing modules in production

5. Defer all dynamic imports to runtime:
   - Never execute at module load time
   - Load in useEffect, event handlers, or async functions
   - This prevents initialization-time circular dependencies
