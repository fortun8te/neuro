═══════════════════════════════════════════════════════════════════════════════
 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

LOW TOOL USAGE (<3 tools)
  → Check if tools are in system prompt
  → Verify token budget isn't too tight
  → Look for abort signal (might be stopping early)

POOR HARNESS SCORE (<6/10)
  → Check errorHandling.ts is imported
  → Verify abort signal threaded through
  → Look for try/catch in tool execution

SLOW EXECUTION (>4 min/question)
  → Check model size (use qwen3.5:4b, not 27b)
  → Verify no sequential operations that should be parallel
  → Check resource usage (CPU, memory)

SERVICE TIMEOUT
  → Restart service: kill + restart in terminal
  → Check logs: tail -f /tmp/wayfayer.log
  → Reduce concurrency if Docker is struggling
